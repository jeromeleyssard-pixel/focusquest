# FocusQuest

Application d'entraînement cognitif adaptatif pour enfants et adolescents TDAH (5–17 ans).

- **FocusQuest Junior** : 5–7 ans — Go/NoGo, 1-Back, DCCS — sessions 10 min, tap uniquement.
- **FocusQuest Standard** : 8–17 ans — CPT-AX, N-Back, Stop-Signal, Task-Switching — sessions 20 min.

Outil d’accompagnement fondé sur des paradigmes validés en neuroscience cognitive. Aucune allégation thérapeutique ; pas de collecte de données personnelles (profil local uniquement).
